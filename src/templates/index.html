Image-to-MusicXML converter with Audiveris

This app converts a image file (music note image in pdf, jpg, png and tif format) into a .mxl-file with Audiveris. I've used the Docker image toprock/audiveris (slightly changed). This conversion process takes several minutes (and i recommend to use only a very small pdf file). So keep calm & be patient...
