Image-to-MusicXML converter with Audiveris

This app converts an image file (music note image in pdf, jpg, png and tif format) into a .mxl-file with Audiveris. I've used the Docker image toprock/audiveris (slightly changed). The conversion process may take several minutes (and It is recommended to use only a very small image file). So keep calm & be patient...
